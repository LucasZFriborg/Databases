# Lab2 – Bokhandel

## SQL Server
Projektet använder SQL Server och SQLAlchemy.

## Databas
Kör `Bokhandel.sql` för att skapa databasen, tabellerna och testdatan.

## Python
Installera dependencies:

pip install sqlalchemy pyodbc

Kör sedan:

python Boksok.py

Programmet frågar efter SQL Server-lösenord vid start.
